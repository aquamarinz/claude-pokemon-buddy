- **M6** ‚Äî `clampDailyTokens` (anti-abuse) is dead code (never imported by production; only its own test). Daily growth is already bounded by `dailyExpCap=100` / `dailyBondCap=6`. Remove the dead module + its test (YAGNI).

**Product note (M7):** This makes KEY long-press a "care/pet the buddy" gesture.
